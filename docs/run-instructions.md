# Frontend run instructions:

## Run instructions for Android:

  • Have an Android emulator running (quickest way to get started), or a device connected.
  • `cd "/Users/victoriatrois/Repositories/iassistente-poc/frontend/iassistente" && npx react-native run-android`
  
## Run instructions for iOS:

  • `cd "/Users/victoriatrois/Repositories/iassistente-poc/frontend/iassistente"`
  • `npx react-native run-ios`
    - or -
  • Open iassistente/ios/iassistente.xcworkspace in Xcode or run `"xed -b ios"`
  • Hit the Run button
    
## Run instructions for macOS:

    • See https://microsoft.github.io/react-native-macos for the latest up-to-date instructions.
